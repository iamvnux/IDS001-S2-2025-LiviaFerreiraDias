Este projeto foi desenvolvido como parte da avaliação P1 da disciplina de Desenvolvimento de Sistemas de Informação.

O sistema simula o funcionamento de uma biblioteca acadêmica, permitindo o cadastro de pessoas (alunos e professores), livros e a realização de empréstimos.

O objetivo principal não é implementar todas as regras de negócio de uma biblioteca real, mas sim demonstrar os conceitos de Programação Orientada a Objetos (POO) em Java, tais como:

Criação de classes e atributos;

- Uso de diferentes níveis de visibilidade;

- Métodos para manipulação de objetos;

- Encapsulamento por meio de getters e setters;

- Herança entre classes;

- Sobrecarga e sobrescrita de métodos.

Dessa forma, o sistema serve como uma aplicação prática dos conceitos estudados em aula, utilizando o NetBeans para organização das classes e execução do programa.
